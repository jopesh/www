Überschriften werden mit `##` bis `######` erstellt. Die erste Überschriftsebene (`#`) wird vom Layout automatisch als Seitentitel gerendert.

Dies ist ein normaler Absatz mit verschiedenen Formatierungen: **fetter Text**, *kursiver Text*, ***fett und kursiv***, ~~durchgestrichener Text~~ und `Inline-Code`.

### TypeScript

```typescript
interface BlogPost {
  title: string;
  description: string;
  pubDate: Date;
  heroImage?: ImageMetadata;
}

const fetchPosts = async (): Promise<BlogPost[]> => {
  const posts = await getCollection("blog");
```
